# merogi-godot-3d-core

A reusable Godot 3D foundation with the core systems that most games need.

Built with **Godot** and **C#**, this project is meant to be a shared starting point for future 3D games. The goal is to create common systems once, keep them modular, and reuse them across multiple projects.

## Planned Features

- Character controller
- Interaction system
- Input system
- Camera controller
- Health and damage
- Inventory and items
- Save / load
- Scene and level management
- State machines
- Event system
- Debug tools
- Basic AI foundations

## Purpose

Instead of rebuilding the same systems for every new project, this repo aims to provide a solid reusable core for common 3D game features in Godot.

## Stack

- Godot
- C#
- 3D game architecture
